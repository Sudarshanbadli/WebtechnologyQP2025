1
Expand WWW, URL, MIME and HTTP.
2 Marks

✅ Detailed Answer

Here are the expansions and definitions of each term:

AbbreviationFull FormDescription
WWW World Wide Web A system of interconnected documents and resources accessed via the Internet using web browsers. Invented by Tim Berners-Lee in 1989, it uses HTML for page structure, URLs for addresses, and HTTP for communication.
URL Uniform Resource Locator The address used to identify and locate resources on the Internet. Format: protocol://domain/path?query. Example: https://www.example.com/page.html
MIME Multipurpose Internet Mail Extensions A standard that classifies file types on the Internet. MIME types tell the browser how to handle different file types. Examples: text/html, image/jpeg, application/pdf
HTTP Hypertext Transfer Protocol The protocol used for transferring web pages over the Internet. It is a request-response protocol between client (browser) and server. HTTPS is the secure version with encryption.
2
What is Hypertext link?
2 Marks

✅ Detailed Answer

A Hypertext Link (or hyperlink) is a reference or pointer to another document, webpage, image, or resource on the Internet. When a user clicks on a hyperlink, the browser navigates to the linked resource.

Key characteristics:

  • Defined using the <a> (anchor) tag in HTML
  • Uses the href attribute to specify the destination URL
  • Can link to web pages, files, email addresses, or specific sections within the same page
  • Can be absolute (full URL) or relative (path from current page) links
HTML <a href="https://www.example.com" target="_blank"> Click here to visit Example </a> <!-- Link to email --> <a href="mailto:info@example.com">Email Us</a>
3
Mention how audio and video can be embedded in HTML.
2 Marks

✅ Detailed Answer

HTML5 introduced the <audio> and <video> tags for embedding multimedia content directly into web pages.

Embedding Audio:

HTML <audio controls autoplay loop> <source src="song.mp3" type="audio/mpeg"> <source src="song.ogg" type="audio/ogg"> Your browser does not support the audio element. </audio>

Embedding Video:

HTML <video width="640" height="480" controls poster="thumb.jpg"> <source src="movie.mp4" type="video/mp4"> <source src="movie.webm" type="video/webm"> Your browser does not support the video element. </video>

Key attributes: controls (shows play/pause buttons), autoplay (auto-plays), loop (repeats), muted (starts without sound), poster (thumbnail for video).

4
Give the syntax of XML structure.
2 Marks

✅ Detailed Answer

XML (eXtensible Markup Language) documents follow a strict structure with the following components:

XML <?xml version="1.0" encoding="UTF-8"?> <!-- XML Declaration --> <!-- Root element (required) --> <rootElement> <!-- Child elements --> <childElement attribute="value"> <subChild>Text content</subChild> </childElement> <anotherChild> <item id="1">First Item</item> <item id="2">Second Item</item> </anotherChild> </rootElement>

Key rules of XML structure:

  • Must have exactly one root element that wraps all other elements
  • All elements must have closing tags
  • Elements are case-sensitive (<Name> ≠ <name>)
  • Elements must be properly nested
  • Attribute values must be enclosed in quotes
  • Starts with an optional XML declaration
5
Expand and define DTD.
2 Marks

✅ Detailed Answer

DTD stands for Document Type Definition.

A DTD defines the legal building blocks (elements, attributes, entities, notations) of an XML or HTML document. It specifies the structure, element names, attributes, and the hierarchy of elements allowed in a document.

Types of DTD:

  • Internal DTD: Defined within the same document
  • External DTD: Defined in a separate file and referenced
HTML/XML <!-- Internal DTD --> <!DOCTYPE note [ <!ELEMENT note (to,from,heading,body)> <!ELEMENT to (#PCDATA)> <!ELEMENT from (#PCDATA)> <!ELEMENT heading (#PCDATA)> <!ELEMENT body (#PCDATA)> ]>
6
What are the levels of style sheets?
2 Marks

✅ Detailed Answer

There are three levels (or types) of CSS style sheets:

TypeScopePriority
Inline Style Sheet Applied to a single element using the style attribute Highest (1000)
Internal (Embedded) Style Sheet Defined in <style> tag within <head> of the HTML document Medium
External Style Sheet Written in a separate .css file and linked using <link> tag Lowest (but reusable)

Priority Order: Inline > Internal > External. If the same property is defined in multiple levels, the higher priority style overrides the lower one.

7
List any four features of JavaScript.
2 Marks

✅ Detailed Answer

Key features of JavaScript include:

  1. Interpreted Language: JavaScript is interpreted (not compiled), meaning it executes line by line directly in the browser without a compilation step, making it fast to test and deploy.
  2. Client-Side Execution: JavaScript runs in the user's browser, enabling dynamic interactions, form validation, and DOM manipulation without server round-trips.
  3. Dynamic Typing: Variables do not need explicit type declarations. The type is determined at runtime, allowing flexible data manipulation.
  4. Event-Driven Programming: JavaScript supports event handling, allowing code to respond to user actions like clicks, key presses, mouse movements, and page loads.
  5. Object-Oriented: JavaScript supports object-oriented programming with prototypes, classes, inheritance, and encapsulation.
  6. Cross-Platform: Works across all major browsers and operating systems (Windows, macOS, Linux, mobile).
8
What are general syntactic characteristics of JavaScript?
2 Marks

✅ Detailed Answer

General syntactic characteristics of JavaScript:

  • Case-Sensitive: Variables, functions, and keywords are case-sensitive (myVar ≠ myvar)
  • Statements end with semicolons (;) — though JavaScript has Automatic Semicolon Insertion (ASI), it is recommended to use them
  • Variables are declared using var, let, or const
  • Blocks are defined using curly braces { }
  • Comments: Single-line with // and multi-line with /* */
  • Whitespace is ignored — spaces, tabs, and newlines do not affect execution
  • Loose typing — the same variable can hold different data types at different times
JavaScript // Variable declaration let name = "John"; // String let age = 25; // Number const PI = 3.14; // Constant // Function declaration function greet(name) { return "Hello, " + name; }
9
Define event handling and event handler.
2 Marks

✅ Detailed Answer

Event Handling is the mechanism in JavaScript that allows a program to detect and respond to user actions or system events (like clicks, key presses, page loads, etc.).

Event Handler is a function (or method) that is called when a specific event occurs. It contains the code that defines what should happen when the event is triggered.

There are three ways to register event handlers:

  1. HTML event attributes: onclick="myFunction()"
  2. DOM property: element.onclick = myFunction
  3. addEventListener method: element.addEventListener('click', myFunction)
JavaScript // Using addEventListener (recommended) const btn = document.getElementById("myBtn"); btn.addEventListener("click", function() { alert("Button was clicked!"); });
10
Define onfocus and onerror event.
2 Marks

✅ Detailed Answer

onfocus Event: This event is triggered when an element receives focus — typically when a user clicks on a text input field, textarea, or selects it using the Tab key. It is commonly used to highlight the active field or display helpful hints.

HTML + JS <input type="text" id="name" onfocus="this.style.background='yellow'">

onerror Event: This event is triggered when an error occurs while loading an external resource such as an image, script, or stylesheet. It is commonly used to handle broken images or provide fallback content.

HTML + JS <img src="broken.jpg" onerror="this.src='fallback.jpg'"> <!-- For scripts --> <script src="missing.js" onerror="alert('Script failed to load!')"></script>

State the position property and its types in CSS w.r.t. HTML.
2 Marks

✅ Detailed Answer

The CSS position property defines how an element is positioned in a document. There are five values:

ValueDescriptionUses top/right/bottom/left?
static Default value. Elements are positioned according to normal document flow. Offset properties have no effect. No
relative Element is positioned relative to its normal position. The space it occupies is preserved. Yes
absolute Element is positioned relative to its nearest positioned ancestor (or body if none). Removed from normal flow. Yes

12
List the event types related to reacting to mouse click.
2 Marks

✅ Detailed Answer

The main mouse click-related event types in JavaScript are:

  1. click — Triggered when the mouse button is pressed and released on an element
  2. dblclick — Triggered when the mouse button is double-clicked on an element
  3. mousedown — Triggered when the mouse button is pressed down on an element
  4. mouseup — Triggered when the mouse button is released over an element
  5. contextmenu — Triggered when the right mouse button is clicked (context menu)

Order of events: mousedown → mouseup → click → dblclick (if double-clicked)

13
Explain frames in HTML.
5 Marks

✅ Detailed Answer

Frames in HTML allow a web page to be divided into multiple independent sections, where each section (frame) can load and display a different HTML document. Frames were commonly used before CSS layouts became popular.

Types of Frames:

1. Frameset (Horizontal/Vertical Splitting):

HTML <frameset rows="25%,75%"> <frame src="header.html" name="topFrame"> <frameset cols="20%,80%"> <frame src="menu.html" name="menuFrame"> <frame src="content.html" name="contentFrame"> </frameset> </frameset>

2. Inline Frame (iframe):

HTML <iframe src="https://www.example.com" width="600" height="400" name="myFrame" frameborder="1" scrolling="auto"> Your browser does not support iframes. </iframe>

Key Attributes of <frame> tag:

  • src — URL of the document to display in the frame
  • name — Name of the frame (used as target for links)
  • frameborder — Shows or hides frame borders (1 or 0)
  • scrolling — Controls scrollbars (yes, no, auto)
  • noresize — Prevents the frame from being resized

Note: The <frameset> and <frame> tags are obsolete in HTML5. Modern web development uses CSS layouts (Flexbox, Grid) and <iframe> instead.

15
Discuss 'If-else' and 'Switch' statements in JavaScript.
5 Marks

✅ Detailed Answer

1. If-Else Statement:

The if-else statement executes a block of code if a specified condition is true, and another block if the condition is false. It supports multiple conditions using else if.

JavaScript let marks = 75; if (marks >= 90) { document.write("Grade: A"); } else if (marks >= 70) { document.write("Grade: B"); } else if (marks >= 50) { document.write("Grade: C"); } else { document.write("Grade: F (Fail)"); }

2. Switch Statement:

The switch statement evaluates an expression and matches its value to a case clause. It executes the statements associated with that case. The break statement prevents fall-through to the next case.

JavaScript let day = "Monday"; switch (day) { case "Monday": document.write("Start of the week"); break; case "Friday": document.write("Almost weekend!"); break; case "Saturday": case "Sunday": document.write("Weekend!"); break; default: document.write("Weekday"); }

Comparison:

FeatureIf-ElseSwitch
Condition TypeBoolean expressionsEquality comparison
Best ForComplex conditions, rangesMultiple discrete values
PerformanceEvaluates each conditionOften faster for many cases
Defaultelsedefault
16
Explain how events are handled from Button Elements.
5 Marks

✅ Detailed Answer

Buttons in HTML can trigger various events. There are three main approaches to handle events from button elements:

Method 1: HTML Event Attribute (Inline)

HTML <button onclick="showAlert()">Click Me</button> <script> function showAlert() { alert("Button clicked!"); } </script>

Method 2: DOM Property

JavaScript <button id="myBtn">Click Me</button> <script> const btn = document.getElementById("myBtn"); btn.onclick = function() { alert("Button clicked via DOM property!"); }; </script>

Method 3: addEventListener (Recommended)

JavaScript <button id="btn">Click Me</button> <script> const btn = document.getElementById("btn"); btn.addEventListener("click", function(event) { console.log("Clicked!"); event.preventDefault(); }); </script>

Common Button Events:

EventDescription
onclickFired when button is clicked
onmousedownFired when mouse button is pressed
onmouseupFired when mouse button is released
onmouseoverFired when mouse hovers over button
onmouseoutFired when mouse leaves the button
onfocusFired when button gains focus

addEventListener is the recommended approach because it allows multiple event handlers on the same element, supports event capturing/bubbling, and separates JavaScript from HTML (better code organization).

17
With example explain how you move an element in JavaScript.
5 Marks

✅ Detailed Answer

Moving elements in JavaScript can be done by changing their position, margin, transform, or by manipulating the DOM (appending/moving elements).

Method 1: Using CSS position properties

HTML + JavaScript <div id="box" style="position: absolute; width: 100px; height: 100px; background: red; left: 0px; top: 0px;"></div> <button onclick="moveRight()">Move Right</button> <script> let pos = 0; function moveRight() { const box = document.getElementById("box"); pos = pos + 50; box.style.left = pos + "px"; } </script>

Method 2: Using CSS transform

JavaScript function moveElement() { const elem = document.getElementById("myElement"); elem.style.transform = "translateX(200px) translateY(100px)"; }

Method 3: Smooth animation with setInterval

JavaScript function animateMove() { const box = document.getElementById("box"); let leftPos = 0; const id = setInterval(function() { if (leftPos >= 300) { clearInterval(id); } else { leftPos++; box.style.left = leftPos + "px"; } }, 5); }

Method 4: Moving DOM elements (reordering)

JavaScript // Move an element from one parent to another const element = document.getElementById("myDiv"); document.getElementById("newParent").appendChild(element); // Move before a specific sibling const sibling = document.getElementById("sibling"); sibling.parentNode.insertBefore(element, sibling);

b) Drag and Drop Elements in JavaScript

How Drag and Drop Works:

HTML5 Drag and Drop API allows users to drag elements and drop them into target areas. It involves several events:

EventApplies ToDescription
dragstartDraggable elementFired when dragging starts
dragDraggable elementFired continuously while dragging
dragendDraggable elementFired when dragging stops
dragoverDrop targetFired when dragged over target
dragenterDrop targetFired when entering target
dragleaveDrop targetFired when leaving target
dropDrop targetFired when element is dropped

Complete Drag & Drop Example:

HTML + JavaScript <!DOCTYPE html> <html lang="en"> <head> <title>Drag and Drop Example</title> <style> #dragBox { width: 120px; height: 120px; background: #6366f1; color: white; display: flex; align-items: center; justify-content: center; cursor: grab; border-radius: 10px; font-weight: bold; } #dropZone { width: 300px; height: 200px; border: 3px dashed #ccc; margin-top: 20px; display: flex; align-items: center; justify-content: center; border-radius: 10px; color: #999; } #dropZone.hover { border-color: #6366f1; background: rgba(99, 102, 241, 0.1); } </style> </head> <body> <div id="dragBox" draggable="true"> Drag Me! </div> <div id="dropZone"> Drop Here </div> <script> const dragBox = document.getElementById("dragBox"); const dropZone = document.getElementById("dropZone"); // Drag events on the draggable element dragBox.addEventListener("dragstart", function(e) { e.dataTransfer.setData("text/plain", "dragged"); dragBox.style.opacity = "0.5"; }); dragBox.addEventListener("dragend", function() { dragBox.style.opacity = "1"; }); // Drop events on the target dropZone.addEventListener("dragover", function(e) { e.preventDefault(); // Required to allow dropping dropZone.classList.add("hover"); }); dropZone.addEventListener("dragleave", function() { dropZone.classList.remove("hover"); }); dropZone.addEventListener("drop", function(e) { e.preventDefault(); dropZone.classList.remove("hover"); dropZone.innerHTML = "✅ Element Dropped!"; dropZone.style.borderColor = "#10b981"; dropZone.appendChild(dragBox); }); </script> </body> </html>

Key Steps for Drag & Drop:

  1. Set draggable="true" on the element to be dragged
  2. Handle dragstart event to set data using dataTransfer.setData()
  3. Handle dragover on the drop target and call preventDefault() to allow dropping
  4. Handle drop event to process the dropped element
  5. Use dragenter and dragleave for visual feedback

Important: The preventDefault() call in the dragover event handler is critical — without it, the browser will not allow the drop operation.
